# 站点推荐

## GitHub

- [GitHub Status](https://www.githubstatus.com/)：实时监控 GitHub 服务可用性及历史事件。
- [grep.app](https://grep.app/)：快速搜索开源代码。
- [Shields.io](https://shields.io/)：为开源项目生成动态徽章，显示构建状态、版本、许可证等信息。

## 开发人员工具

- [DevDocs](https://devdocs.io/)：整合多种编程语言和框架的 API 文档，支持离线访问和快速搜索。
- [regex101](https://regex101.com/)：在线正则表达式测试、调试、解释，支持多语言语法和常用参考。
- [Learn Git Branching](https://learngitbranching.js.org/)：通过可视化交互游戏学习 Git 分支操作。
- [bootlin](https://elixir.bootlin.com/linux/latest/source)：在线阅读 Linux 源码。
- [carbon](https://carbon.now.sh/)：根据代码生成漂亮的截图。

## Go

- [JSON-to-Go](https://mholt.github.io/json-to-go/)：将 JSON 转换为 Go 结构体。

## 其他

- [TypeRacer](https://play.typeracer.com/)：测试打字速度和准确率。
